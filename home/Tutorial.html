What is RASomate?

RASomate is automation software that automates basic HEC-RAS processes like running hundreds of projects in a sequence while making simple geometric edits.

RASomate v0.2 performs the following three tasks for a list of any number of models all at once:
1. Sets the Bank Points to the 10-year WSEL.
2. Runs the current plan.
3. Exports the output table in default WISE format.

Is it safe to download?

Yes, depending on your operating system and browser you may receive certain messages from your firewall. Please bypass those messages and always allow the program to run. 
Windows cannot recognize the software as it is not a popular download yet and the current version is not signed by a certificate issuing agency.

How does RASomate work?

It uses the HEC-RAS controller and various Python libraries to automate the bank station adjustment process.

Procedure

1. Copy all HEC-RAS models you want to run with this software from your Citrix storage to Local drive.
    1.1 Open Compass File Explorer from Citrix. 
    1.2 Go to your BLE ENG folder. 
    1.3 Copy the ENG folder.
    1.4 Locate the Local Disk C on your local computer within Compass file explorer. 
    1.5 Paste the ENG folder within your desired folder under \\Client\C$\Users\firstname.lastname
Operation Time: 10-15 minutes

2. Open RASomate.exe and wait for 5-10 seconds for the program to start.

3. Based on what version of HEC-RAS you have installed on your computer, enter the information in the first field.
    3.1 HEC-RAS 5.0.5 => RAS505
    3.2 HEC-RAS 5.0.7 => RAS507
    3.3 HEC-RAS 5.0.0 => RAS500
Then click the "Select Version" button.

4. Clear the contents of the second field which says: "Insert the ENG folder path here:"

4. Copy and paste the path of your ENG folder in your local drive location as text in the field we just cleared. 
Then click the "Add Folder" button.

5. Click "Run"

This will take a few minutes to run all the simulations and save the output. No other action is necessary at this time.

6. Open Compass File Explorer and now copy and paste the ENG folder from local drive to your project folder in N drive on the cloud.

Remarks

The user may notice that the models are run in alphabetical order. In the alpha and beta testing phase, the first model in the series will not experience any change of bank stations. This will be rectified in future releases as more user response is collected. In case you face any difficulties in using this model feel free to contact the developer at any time.
